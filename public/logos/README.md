# Brand logos

Add your branded images here. **Any image file whose name ends with `logo`** will be used by the dashboard.

Expected filenames (`.svg` or `.png`):

- **opay-logo** — OPay
- **funita-logo** — Funita Media & Communication
- **dstv-logo** — DStv
- **gotv-logo** — GOtv

Examples: `opay-logo.svg`, `funita-logo.png`, `dstv-logo.svg`, `gotv-logo.png`.

The app looks for `.svg` first, then falls back to `.png` if the SVG fails to load. If no image is found, the brand name is shown as text.
